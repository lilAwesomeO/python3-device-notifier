	for this script to work you must
		1) have nmap installed
		2) have the python3-nmap pip-module installed
		3) in your ini-file point to a valid:
			- network
			- folder(for the logs)
			- set an appropriate size-limit for the log-folder
			- automate with task-scheduler(win) or crontabs(*nix)
